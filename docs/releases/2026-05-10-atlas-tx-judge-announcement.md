# Atlas TX — judge announcement post

Primary live destination: https://atlastexas.org

Short version:

Atlas TX is a map-first Texas public-interest intelligence system built for county-level investigation.

Start at https://atlastexas.org.

From there, Atlas TX lets you:
- explore county maps for water, permits, operators, environmental burden, and mismatch signals
- drill from statewide maps into concrete county and permit evidence
- use the same evidence stack through a local MCP server and repo-local Atlas TX agent skill
- extend the workflow into a field-verification lane through the Atlas TX Capture Android app

Why this matters:
Texas public data is fragmented across agencies, portals, and workflows. Atlas TX turns that fragmentation into one governed evidence surface for humans and AI-native workflows.

What makes Atlas TX different:
- county-map-first, not dashboard-first
- public-record evidence, not opaque black-box scoring
- MCP + agent skill access to the same evidence the site uses
- an additive field-observation lane that stays clearly separate from authoritative public-record signals

Recommended judge path:
1. Open https://atlastexas.org
2. Inspect the county maps and one investigation path on water or permits
3. See the same evidence exposed through the Atlas TX MCP server
4. See how the Atlas TX skill wraps that MCP surface for external agents
5. Optionally end with the Android capture app / live strip-screening demo as the field-verification extension

One-line closer:
Atlas TX is not just a website — it is a live Texas evidence system with a public web product, an MCP surface, agent skills, and a field-verification lane.
